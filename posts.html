1. Pengertian HTML Semantik

HTML semantik adalah penggunaan elemen HTML yang memiliki makna (meaningful) dan tujuan yang jelas, baik bagi browser maupun pembaca. Elemen semantik memberi arti terhadap konten, bukan hanya untuk tampilan visual. Dengan HTML semantik, struktur halaman menjadi lebih rapi, mudah dibaca, dan dapat dipahami oleh mesin pencari (SEO) serta teknologi pembaca layar (screen reader).

Contoh Perbedaan:

  • Non-semantik: < div> dan < span> → hanya sebagai wadah tanpa makna khusus
  • Semantik: < header>, < article>, < footer> → memilki arti jelas sesuai fungsinya

Fungsi HTML Semantik

Pengunaan HTML semantik memiliki beberapa manfaat:

  1. Meningkatkan SEO
    Mesin pencari memahami struktur dan konteks konten dengan lebih baik.
  2. Aksesibilitas Yang Lebih Baik
    Membantu pengguna dengan disabilitas menggunakan pembaca layar untuk memahami halaman.
  3. Konsistensi Kode
    Memudahkan developer untuk membaca dan mengelola kode.
  4. Standar Web Modern
    Mengikuti rekomendasi W3C untuk pembuatan website yang terstruktur.

Kesimpulan

HTML semantik sangat penting untuk membuat website yang terstruktur, mudah diakses, dan ramah mesin pencari. Dengan memanfaatkan elemen-elemen semantik seperti < header>, < main>, < article>, dan < footer>, kode HTML menjadi lebih bermakna serta memudahkan proses pengembangan jangka panjang.
